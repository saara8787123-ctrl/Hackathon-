Lesson 2: CSS Basics

CSS (Cascading Style Sheets) is used to style HTML elements. It controls colors, fonts, spacing, borders, and layout of web pages.

CSS Syntax Example:

selector {
  property: value;
}

Example: p { color: blue; font-size: 16px; } sets paragraph text to blue and font size 16px.

CSS can be applied via Inline, Internal, or External methods.
