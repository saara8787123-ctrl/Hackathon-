Lesson 3: Excel Basics

Microsoft Excel is a spreadsheet tool used for storing, organizing, analyzing, and visualizing data.

Basic Features:

  • Cells: Individual boxes to enter data
  • Rows & Columns: Organize data
  • Formulas: Perform calculations, e.g., =SUM(A1:A5)
  • Charts: Visualize data using Bar, Line, Pie charts
  • Pivot Tables: Summarize large data easily
